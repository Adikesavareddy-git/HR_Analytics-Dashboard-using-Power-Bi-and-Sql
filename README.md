📊 **A Data-Driven Approach to HR Analytics**

Ever wondered what actually drives employee attrition? I recently completed an **HR Analytics project** where I analyzed employee data to uncover the real reasons behind workforce turnover using **SQL and Power BI**.

Through this project, I explored HR data to transform raw employee records into meaningful insights that can support better retention strategies.

### 🔍 Key Insights from the Analysis

**Demographic Trends**
Analyzed a workforce of **1,470 employees** with an **average age of 37**, identifying which **age groups, departments, and education backgrounds** experience higher attrition.

**Gender-Based Attrition Patterns**
The analysis revealed different attrition patterns across gender groups, with **150 male and 87 female employees** leaving the organization. These insights help HR teams design more targeted retention initiatives.

**Department-Level Insights**
Departments such as **R&D and Sales** showed higher attrition counts, indicating areas where HR leaders may need to improve engagement and work environment strategies.

**Job Satisfaction Analysis**
Using advanced SQL queries, I created a **PIVOT analysis** that mapped **Job Satisfaction scores (1–4)** against various **Job Roles**, helping identify which roles require stronger cultural and leadership interventions.

---

### ⚙️ Project Workflow

**1️⃣ SQL – Data Preparation & Analysis**

The SQL script served as the **analytical engine** of this project. It was used to:

• Calculate **core employee metrics** (total employees and average age)
• Filter employees by **education, department, and education field**
• Compute **attrition counts and attrition percentages**
• Segment attrition by **gender and department**
• Perform **advanced grouping using the PIVOT function** to analyze job satisfaction across roles

This stage transformed raw HR data into structured insights ready for visualization.

---

**2️⃣ Power BI – Interactive Dashboard Development**

The Power BI dashboard acted as the **presentation layer**, converting analytical results into a visual storytelling experience.

Key dashboard components include:

• **Interactive visuals** to explore attrition trends
• **Demographic analysis dashboards** for HR decision-making
• **Department-level attrition insights**
• **Custom dashboard design**, including background visuals and layout optimization
• **Data modeling and relationships** that connect SQL-derived metrics to dynamic reports
